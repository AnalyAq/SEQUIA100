# SEQUIA100


Somos estudiantes de la especialidad de Ingienería Metorológica y Gestión de Riesgos Climáticos. 
A través de este repositorio, mostramos la metodología usada en nuestro proyecto final del curso de tecnicas de programación 2, titulado "Análisis de sequías en Piura usando el SPI y proyecciones al 2100"
Podrán encontrar la data empleada y los códigos de acuerdo a la metodología, donde:
AED: Análisis exploratporio de datos
codigo_climatología: Diagramas de cajas para cada estación
SPI_corregido: cálculo del SPI para la data histórica
trabajo_final: Muestra las proyecciones al 2100
